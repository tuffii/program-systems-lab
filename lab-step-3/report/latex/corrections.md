# Исправленные участки отчета lab-step-3

## Проверка адресов

В разделе с пакетом объектных карт учтено начальное смещение на `0x02`: команда `BALR @RBASE,0` размещена по адресу `0x00` и загружает в базовый регистр адрес следующей команды, то есть `0x02`. Поэтому для всех операндов, адресуемых через `@RBASE`, в объектный код записывается не полный относительный адрес метки, а смещение:

```text
смещение = относительный адрес метки - 0x02
```

Например:

```text
@BUF:  0x40 - 0x02 = 0x3E
B:     0x36 - 0x02 = 0x34
@FIVE: 0x3C - 0x02 = 0x3A
C:     0x3A - 0x02 = 0x38
A:     0x34 - 0x02 = 0x32
@TRUE: 0x2A - 0x02 = 0x28
@SAVE: 0x2E - 0x02 = 0x2C
```

## Исправления в таблице ручного ассемблирования
